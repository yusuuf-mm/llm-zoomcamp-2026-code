# LLM Zoomcamp 2026

Code, notebooks, and homework for the [LLM Zoomcamp 2026](https://github.com/DataTalksClub/llm-zoomcamp) cohort by [DataTalks.Club](https://datatalks.club).

## Modules

- `01-agentic-rag` — RAG from scratch, indexing, chunking, and agentic tool calling.
- `homework` — structured homework notebook with all submitted answers.

## Setup

```bash
cp .env.example .env   # add your OPENROUTER_API_KEY
uv sync
```

## Run

```bash
jupyter lab
```
