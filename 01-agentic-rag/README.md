# 01-agentic-rag

Module 1 of LLM Zoomcamp 2026: building a RAG system from scratch and making it agentic.

## Contents

- `notebook.ipynb` — lab notebook with environment setup, LLM client initialization, knowledge-base loading via `gitsource`, indexing with `minsearch`, and an agentic tool-calling loop.

## Run

```bash
uv add minsearch gitsource openai python-dotenv
jupyter lab 01-agentic-rag/notebook.ipynb
```
